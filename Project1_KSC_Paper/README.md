# Deep Learning Model Optimization for Real-Time Healthcare Service Congestion Prediction

## Overview
A research project proposing a Synthetic Time Series generation method to overcome structural limitations of public emergency room data, published at KSC 2025.

## Tech Stack
Python, XGBoost, Random Forest, CNN-LSTM, Pandas, Matplotlib, TensorFlow

## Key Contributions
- Proposed Synthetic Time Series algorithm reflecting seasonality, volatility, and continuity
- Integrated weather (ASOS) and public holiday API for enhanced feature engineering
- Applied lag features, rolling statistics for time series modeling
- Compared CNN-LSTM, XGBoost, RandomForest models
- XGBoost achieved best performance (MAE 59.38, RMSE 68.30)

# 실시간 의료 서비스 혼잡도 예측을 위한 딥러닝 모델 최적화 연구

## 개요
공공 응급실 데이터의 구조적 한계를 극복하기 위한 Synthetic Time Series 생성 기법을 제안한 연구 프로젝트. 2025 한국소프트웨어종합학술대회(KSC) 게재.

## 기술 스택
Python, XGBoost, Random Forest, CNN-LSTM, Pandas, Matplotlib, TensorFlow

## 주요 기여
- 계절성, 변동성, 연속성을 반영한 Synthetic Time Series 알고리즘 제안
- 기상청 ASOS 및 공휴일 API 통합으로 피처 강화
- 지연 피처, 이동 통계 등 정교한 특성 공학 적용
- CNN-LSTM, XGBoost, RandomForest 모델 비교 분석
- XGBoost 최우수 성능 달성 (MAE 59.38, RMSE 68.30)
